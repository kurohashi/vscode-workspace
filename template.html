If you are using a native android app, then the following few steps will help you in implementing fibotalk in your app.

  1. The fibotalk module uses internet, so make sure the app has internet access permission. The AndroidManifest.xml should have,
    <uses-permission android:name="android.permission.INTERNET" />

  2. Paste the following code in all you activity layout XMLs:
    <com.fibo.FiboAPI
       android:layout_width="match_parent"
       android:layout_height="match_parent"
       android:id="@+id/fibo"
       appid=”your_appid” />

  3. Get “your_appid” from “ https://app.fibotalk.com/app/help/quick-help ” after logging in

  4. Now, in your activity code, import two classes:
    import com.fibo.Fibo;
    import com.fibo.FiboAPI;

  5. In the activity class, declare a private member variable:
    private Fibo fibo;

  6. In the onCreate(), initialize the fibo class member:
    fibo = new Fibo((FiboAPI) findViewById(R.id.fibo));

The basic implementation is done.




Send user info to fibotalk

fibo.set(“userInfo”, “{user: {userId: ‘mandatory userId’, name: ‘Your name’, group: ‘General’, phone: ‘9999999999’}}”);

Here, the details provided are (example):

  • name: “Your name will come here”.
  • group: “Your organisation name comes here”.
  • phone: “Your phone number comes here”.
  • userId: (mandatory) “your user identification”.






Send account info to fibotalk

fibo.set(“userInfo”, “{user: {userId: ‘mandatory userId’}, account: {accountId: ‘mandatory accountId’, accName: ‘Your account name’}}”);

Here, the details provided are (example):

  • accName: “Your name will come here”.
  • accountId: (mandatory) “your account identification”.
  • userId: (mandatory) “your user identification”.




Send events to fibotalk

You can send events to fibotalk to track the activity of the user.

  • Click event: onClick of a button send click_event like,

fibo.set(“click_event”, “button text”);

  • Page open: onStart()/onResume() in an activity, send the page_open like,

fibo.set(“page_open”, “activity name”);

  • Signup event: when a signup in your app is successful, send signup event like,

fibo.set(“signup”, “{user: {userId: ‘mandatory unique user identification’, name: ‘name of the user’}}”);
